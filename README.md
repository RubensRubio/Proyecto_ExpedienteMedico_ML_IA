# Proyecto MVP – Sistema de Consulta y Búsqueda de Pacientes

Pasos rápidos:
1. Crear y activar venv: `python3 -m venv venv` / `source venv/bin/activate`
2. Instalar dependencias: `pip install -r requirements.txt`
3. Convertir `data/sample_data.csv` a `data/sample_data.xlsx` (ver instrucciones)
4. Ejecutar: `python src/app.py`
